React Notes

/* React is a JavaScript library for building user interfaces. */
  • React allows developers to create large web applications that can change data, without reloading the page.
  • React is known for its virtual DOM feature, which optimizes rendering and improves performance.
  • React can be used with other libraries or frameworks, such as Redux for state management.
  • in coding languages Imperative programming focuses on how to achieve the result by explicitly telling the computer what to do, while Declarative programming focuses on what the result should be.
  • SPA: Single Page Application: Browser loads a single HTML page .and dynamically updates content as the user interacts with the app.
  • HTML/CSS/SQL are Declarative languages.
  • Java/C#/C and vanilla javascript are imperative languages.
